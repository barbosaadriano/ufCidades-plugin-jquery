Plugin - Estados & Cidades do Brasil


Como utilizar

Requisitos :

  • jQuery
  • Arquivo JSON com cidades e estados 

Organização dos diretórios (recomendado)
/index.html
/js/jquery-X.X.X.js
/js/ufCidades.plugin.jquery.js
/data/estados-cidades.json

HTML

//Marcar elemento que servirá de container com um seletor ex.: form id="frm-teste" ou div class="foo"

JQuery

$(document).ready(function(){ $('#frm-teste').ufCidades(); });

Opções

é possível passar parametro para ufCidades(), alterando algumas opções. 

// As opções são informadas seguindo o padrão JSON chave:valor conforme exemplo a seguir.
// caso não seja passado o parâmetro é assumido o default.
{
	"urlJson":"./data/estados-cidades.json",
	"captionEstados":"Estados",
	"captionCidades":"Cidades",
	"nomeCampoEstado":"estado",
	"nomeCampoCidade":"cidade",
	"defaultMessageEstado":"Selecione um estado!",
	"defaultMessageCidade":"Selecione uma cidade!",
	"selectClasses":"form-control"
}

Exemplo de passagem de parâmetro

			
	$('#frm1-teste').ufCidades({
		"captionEstados":"Lista de estados - caption personalizado",
		"captionCidades":"Caption de lista de cidades",
		"nomeCampoEstado":"foo",
		"nomeCampoCidade":"bar",
		"defaultMessageEstado":"Selecionaaaaa!!",
		"defaultMessageCidade":"Selecionaaaaa por favor!",
		"selectClasses":"form-control"});
	});
